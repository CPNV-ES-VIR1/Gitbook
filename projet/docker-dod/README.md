# Docker Dod

## Intention

Voici un ensemble de documents relatant les principes à suivre lorsque vous déployerez des conteneurs en production. Il est conçu comme une "checklist" vous permettant d'évaluation votre installation et de vos pratiques.

Ce document permet également de comprendre pourquoi un simple "apt install docker" ne peut pas être une solution pérenne en production.
