# Evaluation Théorique

## Consignes et déroulement

### Théorique, conception

| Critère               | Description                                                                                                                                                                                                                                                                                                                                                                                           |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Moyens auxiliaires    | Aucun                                                                                                                                                                                                                                                                                                                                                                                                 |
| Forme d'examen        | Ecrite (office 365 - form)                                                                                                                                                                                                                                                                                                                                                                            |
| Forme collaborative   | Individuelle                                                                                                                                                                                                                                                                                                                                                                                          |
| Critères d'évaluation | <p></p><ul><li>expliquer et détailler les concepts clés de la virtualisation</li><li>mettre en évidence les différences entre la virtualisation classique et la conteneurisation</li><li>comprendre les principes de bases de la contenerisation ainsi que la gestion d'image et des conteneurs.</li><li>pouvoir décrire les avantages des deux architectures monolithic et micro-services.</li></ul> |
| Pondération           | 30% de la note du module                                                                                                                                                                                                                                                                                                                                                                              |
| Durée                 | 45 minutes                                                                                                                                                                                                                                                                                                                                                                                            |

{% hint style="warning" %}
Lors de l'épreuve, vous devrez répondre aussi bien à des questions en liens avec la théorie et les concepts vus en classe, tout comme le contenu des laboratoires.
{% endhint %}

{% hint style="danger" %}
Le framework backend Spring ne sera pas évalué dans cette épreuve. C'est le projet de module qui permettra de voir si vous avez acquis les compétences en développement.
{% endhint %}
